# apiinfo

#### 介绍
PHP框架API文档生成工具, 无需生成静态文档. 动态生成. 目前完成了基于Laravel的实现. 

#### 软件架构
软件架构说明


#### 安装教程

1.  Composer安装

>我本人的开发环境是PHP7.3, 目前还没有做向下兼容的测试

```
composer require spool/apiinfo
```
>运行这个命令添加资源和配置
```
php artisan vendor:publish --tag=apiinfo
```
打开 http://localhost/apiinfo/ 即可看到演示页面
